最近發現..係時候用下腦記下D Visual Studio Shortcut
有時候覺得其實用 mouse 都可以..
但係用shortcut key 有時真係方便好多

**Remove and Sort Using**
`Ctrl + R Ctrl + G`

**Comment the selected line**
`Ctrl K + Ctrl + C`

**Uncomment the selected line**
`Ctrl K + Ctrl + U`

Hope you find it useful
