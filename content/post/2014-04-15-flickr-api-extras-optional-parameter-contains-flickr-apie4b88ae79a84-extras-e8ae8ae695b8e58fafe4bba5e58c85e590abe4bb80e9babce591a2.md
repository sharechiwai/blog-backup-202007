最近更新自己網頁上的 Album Page 希望可以令到他更完善  
所以開始仔細地看**Flickr API**了  
發現之前忽略了的一些變數..  
Flickr API上很多時候都會有一個叫做**Extras** 或 **primary\_photo\_extras** 的parameter  
<img class="alignnone" src="https://i1.wp.com/farm6.staticflickr.com/5342/14108899082_4bd8ddb5ce_d.jpg?resize=500%2C315" alt="Flickr API primary_photo_extra" width="500" height="315" data-recalc-dims="1" />  
如果使用了他..你便可以在**Flickr API** 上取得更多的資料了  
在這個變數內可以加入以下的 variable

  * **license**
  * **date_upload**
  * **date_taken**
  * **owner_name**
  * **icon_server**
  * **original_format**
  * **last_update**
  * **geo**
  * **tags**
  * **machine_tags**
  * **o_dims**
  * **views**
  * **media**
  * **path_alias**
  * **url_sq**
  * **url_t**
  * **url_s**
  * **url_m**
  * **url_o**

這篇網誌主要是給自己做一個Notes所以質素比較差.. 呵呵

Hope you find it useful
