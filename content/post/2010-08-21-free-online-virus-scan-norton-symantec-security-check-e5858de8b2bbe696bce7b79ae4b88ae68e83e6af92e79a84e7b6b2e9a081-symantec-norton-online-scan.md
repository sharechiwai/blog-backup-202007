如果有機會的話, 我會發多一些文章關於線上掃毒的網頁,  
我覺得久不久用一些線上掃毒 的網頁來檢查電腦有沒有中毒/或安裝了一些間諜軟體又不知道&#8230;ETC  
因為即使電腦上已經安裝了防毒軟件..但都可能會有一些病毒可能不被這防毒軟件認到的, 所謂..[百密一疏, 很小有東西是完美的]&#8230;

今天想和大家介紹的是Symantec Security Check [Norton Online Scan],  
我覺得這個線上掃毒需要的時間比較其他的快, 所以如果覺得電腦沒有什麼大問題的時候我都喜歡用他來作線上掃毒

**使用方法:**  
大家可以到以用IE 來開啟以下的Link  
<http://security.symantec.com/sscv6/WelcomePage.asp>  
因為他需要使用ActiveX來做Virus Scan 的.  
[<img title="NortonOnlineScan1" src="https://i1.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/nortononlinescan1.jpg?resize=625%2C496" alt="" width="625" height="496" data-recalc-dims="1" />](https://i1.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/nortononlinescan1.jpg)

以下是使用這個線上掃毒的步驟,  
1)按下 &#8220;Continue to Symantec Security Check&#8221;來開始線上掃毒  
2) IE會要求你執行 一些 IE 的Add-on [外掛的] 例如:MSXML3.0  
大家便要按這個黃色bar 去選擇&#8221;Run Add-on /執行&#8221;  
[<img class="alignnone size-full wp-image-452" title="NortonInsallMSXML3.0" src="https://i2.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/nortoninsallmsxml3-0.jpg?resize=625%2C492" alt="" width="625" height="492" data-recalc-dims="1" />](https://i2.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/nortoninsallmsxml3-0.jpg)

3)之後便會有一個視窗要問你執行這個Add-on 嗎 [是要執行的]  
[<img title="InstallAddon" src="https://i1.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/installaddon.jpg?resize=474%2C216" alt="" width="474" height="216" data-recalc-dims="1" />](https://i1.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/installaddon.jpg)  
4) 選擇&#8221;I accept/我同意&#8221; 服務條款  
[<img class="alignnone size-full wp-image-453" title="NortonLicense" src="https://i2.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/nortonlicense.jpg?resize=625%2C547" alt="" width="625" height="547" data-recalc-dims="1" />](https://i2.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/nortonlicense.jpg)  
5)和上面一樣還有其他東西要要裝才可以執行線上掃毒  
這次是&#8221;Symantec Security Check 的元件&#8221;, 所以都是按這個黃色bar 去選擇&#8221;Run Add-on /執行&#8221;  
[<img class="alignnone size-full wp-image-451" title="installNortonEngine" src="https://i1.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/installnortonengine.jpg?resize=625%2C550" alt="" width="625" height="550" data-recalc-dims="1" />](https://i1.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/installnortonengine.jpg)  
6)之後便會出現一個視窗叫你嘗試再這到這個重新連線到這個網頁  
[<img class="alignnone size-full wp-image-454" title="NortonWarning" src="https://i0.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/nortonwarning.jpg?resize=445%2C227" alt="" width="445" height="227" data-recalc-dims="1" />](https://i0.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/nortonwarning.jpg)  
7)安裝 &#8220;Symantec Security Check Component /Symantec Security Check的元件&#8221;  
[<img class="alignnone size-full wp-image-449" title="InstallAddon" src="https://i1.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/installaddon.jpg?resize=474%2C216" alt="" width="474" height="216" data-recalc-dims="1" />](https://i1.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/installaddon.jpg)  
8) 下載ActiveX 控制中  
[<img class="alignnone size-full wp-image-450" title="InstallingActiveX" src="https://i1.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/installingactivex.jpg?resize=625%2C436" alt="" width="625" height="436" data-recalc-dims="1" />](https://i1.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/installingactivex.jpg)  
9)安裝 &#8220;Virus Definitions /病毒定義元件&#8221;  
[<img class="alignnone size-full wp-image-456" title="VirusDefination" src="https://i0.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/virusdefination.jpg?resize=474%2C216" alt="" width="474" height="216" data-recalc-dims="1" />](https://i0.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/virusdefination.jpg)  
10) 終於開始掃毒了 在這個 時候請勿關掉這個IE, 待他可以進行掃毒  
[<img class="alignnone size-full wp-image-455" title="StartScanVirus" src="https://i1.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/startscanvirus.jpg?resize=625%2C436" alt="" width="625" height="436" data-recalc-dims="1" />](https://i1.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/startscanvirus.jpg)  
11) 掃毒完畢後便會看到掃毒報告, 這次是我第一次找到病毒/廣告程式&#8230;很可惜&#8230; Norton 沒有提供一個很方便的功能去清除這個病毒/廣告程式, 所以 我便要跟據他的 建議在他的網站上尋找解決這病毒/廣告程式的方法, 之後跟隨着解決方法去清除這病毒/廣告程式  
[<img class="alignnone size-full wp-image-460" title="VirusFound" src="https://i2.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/virusfound.jpg?resize=625%2C514" alt="" width="625" height="514" data-recalc-dims="1" />](https://i2.wp.com/oldblog.sharechiwai.com/wp-content/uploads/2010/08/virusfound.jpg)  
由於覺得比較煩..所以我決定直接刪除這個檔案&#8230;因為我知道這個檔案是不重要的
