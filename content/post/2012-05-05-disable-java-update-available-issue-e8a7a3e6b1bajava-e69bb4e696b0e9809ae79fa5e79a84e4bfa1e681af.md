最近有朋友說電腦久不久便會出現 &#8220;**Java Update Available**&#8221; 的Popup  
<img src="http://api.photoshop.com/v1.0/accounts/aa9037104a014abbb11ad4bd58324b91/assets/e4ada2ca561a41a09648236ddd2dc963" alt="Java Update Available" width="305" height="112" />  
感到十分麻煩..

這是因為電腦上設定了&#8221;自動檢查更新&#8221;的關係

**解決方法**:  
開啟**Java** 這個程式的&#8221;**內容**&#8221;  
在可以 **Task Bar**上的**Java** icon 上按右鍵..選擇&#8221;**Properties**/**內容**&#8221;  
<img src="http://api.photoshop.com/v1.0/accounts/aa9037104a014abbb11ad4bd58324b91/assets/6392cc2cc3974cff93896d6511b9e1a0" alt="Java Update icon on Task Bar" width="222" height="221" /> 

或者可以在&#8221;**Control Panel**/**控制台**&#8221; 上或出&#8221;**Java**&#8220;程式的Icon 按一下  
<img src="http://api.photoshop.com/v1.0/accounts/aa9037104a014abbb11ad4bd58324b91/assets/9a066c2baed84d228edc07201aaef279" alt="Control Panel Item - Java" width="816" height="607" />  
在&#8221;**Java Control Panel**/**Java 程式控制台**&#8221; 上選擇 &#8220;**Update**/**更新**&#8220;分頁  
取消劃勾 Untick&#8221;**Check for Updates Automatically**/**自動檢查更新**&#8221; 這個方塊/Checkbox  
<img src="http://api.photoshop.com/v1.0/accounts/aa9037104a014abbb11ad4bd58324b91/assets/072a690c600d4104ba5885fb1ae9752b" alt="Java Control Panel - Check for Updates Automatially" width="458" height="513" />  
之後會出現一個彈出信息/Popup Message 說  
&#8220;**You have chosen to never check for updates and may miss future security updates**&#8220;/&#8221;**你選擇了從不檢查更新，並可能錯過未來的安全更新**&#8221;  
<img src="http://api.photoshop.com/v1.0/accounts/aa9037104a014abbb11ad4bd58324b91/assets/259e612a22fe41bbb28b2d18c3e3c06b" alt="You have chosen to never check for updates and may miss future security updates" width="535" height="517" />  
之後大家可以選擇&#8221;**Check Monthly**/**每月檢查**&#8221; 或 &#8220;**Never Check**/**永不檢查**&#8221;

選擇了&#8221;**Never Check**/**永不檢查**&#8221; 電腦上便不會再久不久彈出一些叫你更新**Java**的信息了

當然如果大家是會更用Java的話..  
可以久不久自行檢查有沒有新的更新

大家可以在&#8221;**Update tab**/**更新分頁**&#8220;上按 &#8220;**Update Now**/**馬上更新**&#8221;  
便可以開始更新了

Hope you find it useful
